# 🔐 Secure Password Toolkit

A beginner-friendly cybersecurity project built with **Python** and **CustomTkinter**. This desktop application allows users to generate secure passwords and evaluate their strength using common password security principles.

---

## ✨ Features

### 🔑 Password Generator
- Generate random passwords
- Adjustable password length (4–16 characters)
- Include:
  - ✅ Uppercase letters
  - ✅ Lowercase letters
  - ✅ Numbers
  - ✅ Symbols
- Generate Again button
- Copy Password button

### 🛡️ Password Strength Checker
- Analyze password strength
- Real-time strength checking
- Entropy score calculation
- Password strength progress bar
- Suggestions to improve weak passwords
- Show/Hide password option

### 🎨 User Interface
- Modern GUI using CustomTkinter
- Light/Dark Theme Toggle
- Clean two-panel layout

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Secure_Password_Toolkit.git
```

2. Open the project

```bash
cd Secure_Password_Toolkit
```

3. Install CustomTkinter

```bash
pip install customtkinter
```

4. Run the application

```bash
python main.py
```
